Introdução

Este trabalho consiste na análise estrutural e algorítmica de um grafo atribuído, com o objetivo de extrair propriedades fundamentais, executar algoritmos clássicos e investigar fenômenos como small‑world, lei de potência e robustez. O relatório final deve documentar todas as medidas computadas, justificar eventuais ausências e incluir uma tabela resumo. O dataset utilizado nesse experimento foi disponibilizado pelo SNAP - Stanford Network Analysis Project

Este livro foi originalmente criado e publicado no Quarto a fim de fornecer um exemplo de seu uso. Acessem o link : https://carolbalbs.github.io/tp-final. Este site é uma adaptação do código-fonte original para o sistema de publicação do framework.

Começando

O currículo de visualização pode ser usado online ou em seu computador local. Você pode visualizar e interagir com os gráficos diretamente neste Jupyter Book. Se desejar modificar o código, você tem algumas opções diferentes:

  • Para ler notebooks JavaScript online usando Observable, navegue até a página “Observable” acima e clique no notebook correspondente.
  • Para ler notebooks Python online usando Colab, clique na seção correspondente neste livro, passe o mouse sobre o pequeno foguete no topo da página e selecione “Colab” no menu.
  • Para ler notebooks Python localmente, siga as instruções abaixo.

Instalação Local

  1. Instale Altair e um ambiente de notebook. As versões mais recentes destes notebooks usam Altair versão 4.
  2. Baixe os notebooks na página de releases. Normalmente, você desejará usar a versão mais recente. (Se você deseja usar notebooks para Altair versão 3, baixe o Altair v3.2 release.)
  3. Abra os notebooks em seu ambiente de notebook local. Por exemplo, se você tiver JupyterLab instalado (v1.0 ou superior é necessário), execute jupyter lab no diretório contendo os notebooks.
