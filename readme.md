## Repositório e Instruções de Execução

O código-fonte completo está disponível em: <https://github.com/seu-usuario/seu-repositorio>

O repositório contém:

- **scripts/** — códigos Python utilizados nas análises 
- **scripts/logs** — arquivos de log dos scripts executados no servidor
- **data/** — conjunto de dados tratados utilizados
- **data/images** — conjunto graficos gerados na analise
- **README.md** — instruções detalhadas de como executar o código e reproduzir os resultados
